### `> whoami`

I build backend and real-time infrastructure for things that need to feel instant — matchmaking, live game state, and low-latency systems under the hood of products people actually play.

🕊️ **Founder**, building three products at once:

1️⃣ **[Lethal Hive](https://lethalhive.com)** — esports/matchmaking platform (CS2, R.E.P.O., more to come), built around Steam Game Coordinator integration, Overwolf GEP overlays, and a NestJS/Redis/NATS backend designed to scale.
