### `> whoami`

I build backend and real-time infrastructure for things that need to feel instant — matchmaking, live game state, and low-latency systems under the hood of products people actually play.

Currently heads-down on **[Lethal Hive](https://lethalhive.com)**, an esports/matchmaking platform (CS2, R.E.P.O., more to come) — handling everything from Steam Game Coordinator integration and Overwolf GEP overlays to a NestJS/Redis/NATS backend built to scale.

Also contributing backend work to **NoorScanner**, an AI-assisted radiology platform — a different kind of "real-time matters," same engineering mindset.

```
🎮  esports infra      ⚡ real-time systems      🛰  game integrations (Steam / Overwolf / GEP)
```
